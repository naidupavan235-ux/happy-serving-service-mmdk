# Happy Serving Services

A professional marketing landing page for **Happy Serving Services** — an event staffing company providing trained serving boys for weddings, parties, corporate events, and more across Andhra Pradesh.

## Key Technologies

- **TanStack Start** (React-based SSR framework)
- **Tailwind CSS v4** — utility-first styling
- **Netlify** — hosting and deployment

## Running Locally

```bash
npm install
npm run dev
```

The dev server starts at [http://localhost:3000](http://localhost:3000).

To use all Netlify features locally (edge functions, forms, etc.):

```bash
netlify dev --port 8889
```
